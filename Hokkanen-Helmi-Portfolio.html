Projects

Morse Code Puzzle Game using Unity

A 3D puzzle game developed in Unity as a group project for an interactive prototyping course. We used Miro to assign tasks for each team member and keeping track of the progress. I designed the environments that included the rooms and hallways using Unity Asset Store items we agreed upon within the group. In addition I created UI screens, chose typography, and built the Morse input interface for players. I was also in charge for the morse puzzle logic that made it so that each door had increasing difficulty levels on the morse puzzle words generated to the player.

Smartwatch Application with Light & Dark Mode UI

A smartwatch application prototype for a hiking app created for a user interface design course. I was in charge of implementing our groups design choices for the smartwatch views and making the interactive prototype of our smartwatch application using Figma. We used Mural weekly to present our progress in class.
